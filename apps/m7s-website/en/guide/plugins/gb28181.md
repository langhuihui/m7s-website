# GB28181 Plugin

This plugin provides services for the SIP server and streaming media server, and can capture the streams of NVRs and cameras to m7s, allowing access to recording data and viewing recorded videos. It can also control camera rotation, zoom, and other settings.

## Plugin Address

github.com/Monibuca/plugin-gb28181

## Plugin Introduction

```go
import (
_ "m7s.live/plugin/gb28181/v4"
)
```

## Default Plugin Configuration

```yaml
gb28181:
  autoinvite:     true #automatically initiates an invite, to get the stream immediately after receiving device information from the server(SIP)
  position:
    autosubposition: false #whether to automatically subscribe for positioning information
    expires: 3600s #subscription period (in seconds), default 3600
    interval: 6s #subscription interval (in seconds), default 6
  prefetchrecord: false 
  udpcachesize:   0 #UDP cache size, default is 0, not enabled. Only when TCP is off and the cache size is greater than zero, UDP cache will be enabled.
  sipnetwork:     udp
  sipip:          "" #SIP server address, automatic adaptation to device network segment
  sipport:        5060
  serial:         "34020000002000000001"
  realm:          "3402000000"
  username:       ""
  password:       ""
  
  registervalidity:  60s #registration validity period
  
  mediaip:          "" #media server address, automatic adaptation to device network segment
  mediaport:        58200 #media server port, used to receive device streams
  medianetwork:     tcp
  mediaportmin:     0 #minimum value of the media server port range, enabling the port range mode after setting
  mediaportmax:     0 #maximum value of the media server port range, enabling the port range mode after setting

  removebaninterval: 10m #remove the blacklist of devices that have registration failure regularly, in seconds, default 10 minutes (600 seconds)
  loglevel:         info
```

**If the port range is configured, the port range mechanism will be used, and each stream corresponds to a port.**

**If some cameras do not have a username setting, the camera will use its own national standard ID as the username, and m7s will ignore the username of the camera and the username configuration.**
If the device is configured with an incorrect username and password, after three consecutive reported errors, m7s will record the device ID and prohibit the device from registering within 10 minutes.

## Plugin Functions

### Use the SIP protocol to register NVR or other GB28181 devices

- When the server starts, it will automatically listen for the SIP protocol port. When a device registers, the device information will be recorded and the device can be seen in the UI list.
- Send the Catalog command regularly to query device directory information and obtain channel data or sub-devices.
- Send the RecordInfo command to query the device for recording data.
- Send the Invite command to obtain the live video or recorded video of the device.
- Send the PTZ command to control the camera PTZ.
- Automatically synchronize device location.

### Accept media streams from GB28281 devices as a streaming media server

- When an **instant** video stream is invited from the device, the corresponding stream will be created in m7s, and StreamPath is composed of the device number and channel number, i.e., [device number]/[channel number]. If there are multiple levels, the channel number is the number of the last level.
- When an **recorded** video stream is invited from the device, the StreamPath is composed of the device number, channel number, and the start and end time of the recording, i.e., [device number]/[channel number]/[start time]-[end time].

### How to Set UDP Cache Size

Analyze the RTP by capturing packets via Wireshark, and then see how many packets can be ordered.

## API

### List All GB28181 Protocol Devices

`/gb28181/api/list`
The device structure is as follows:

```go
type Device struct {
	ID              string
	Name            string
	Manufacturer    string
	Model           string
	Owner           string
	RegisterTime    time.Time
	UpdateTime      time.Time
	LastKeepaliveAt time.Time
	Status          string
	Channels        []*Channel
	NetAddr         string
}
```

> According to the Golang rules, variables starting with lowercase letters will not be serialized.

### Pull Video Streams from Devices

`/gb28181/api/invite`

| Parameter  | Required | Description                                          |
| ---------- | -------- | ---------------------------------------------------- |
| id         | Yes      | Device ID                                            |
| channel    | Yes      | Channel number                                       |
| startTime  | No       | Start time in pure digital Unix timestamp            |
| endTime    | No       | End time in pure digital Unix timestamp              |

Returns 200 for success.

### Stop Pulling Stream from Devices

`/gb28181/api/bye`

| Parameter  | Required | Description                                          |
| ---------- | -------- | ---------------------------------------------------- |
| id         | Yes      | Device ID                                            |
| channel    | Yes      | Channel number                                       |

### Send Control Commands

`/gb28181/api/control`

| Parameter  | Required | Description                                          |
| ---------- | -------- | ---------------------------------------------------- |
| id         | Yes      | Device ID                                            |
| channel    | Yes      | Channel number                                       |
| ptzcmd     | Yes      | PTZ control command                                  |

### Query Recording

`/gb28181/api/records`

| Parameter  | Required | Description                                                |
| ---------- | -------- | ---------------------------------------------------------- |
| id         | Yes      | Device ID                                                  |
| channel    | Yes      | Channel number                                             |
| startTime  | No       | Start time, string format: 2021-7-23T12:00:00 (UTC + 08:00) |
| endTime    | No       | End time, string format same as above                       |

### Mobility Position Subscription

`/gb28181/api/position`

| Parameter  | Required | Description             |
| ---------- | -------- | ----------------------- |
| id         | Yes      | Device ID               |
| expires    | Yes      | Subscription period (s) |
| interval   | Yes      | Subscription interval(s)|
